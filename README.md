# toy-record-service

## Goal
**Job-fit experiment (Backend vs Infrastructure vs Frontend)**

이 프로젝트는 포트폴리오용이 아니라,  
**30일 동안 직접 구현하며 개발 직무 적합성을 판단하기 위한 실험 프로젝트**다.

---

## Project Overview

- Language: **C++**
- HTTP Server: **cpp-httplib**
- Storage: **In-memory (vector)**
- Frontend: **Minimal (JS fetch 예정)**
- Environment: **Linux 배포 및 운영 실험 예정**
